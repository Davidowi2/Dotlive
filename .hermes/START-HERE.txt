================================================================================
                           WHAT WAS BROKEN
================================================================================

PROBLEM: Every create/update/delete returned HTTP 500
- Users couldn't create ventures
- Users couldn't create pitch decks
- Users couldn't create feed posts
- Users couldn't do ANY database write operation

ROOT CAUSE: Duplicate bootstrap migrations
- Server created the same tables TWICE
- In TWO different locations
- With CONFLICTING schemas
- When routes tried to write, schemas didn't match → 500 error

================================================================================
                            WHAT WE FIXED
================================================================================

SOLUTION: Removed duplicate bootstrap code
- Cleaned up health check endpoint (removed table creation)
- Removed conflicting table definitions
- Consolidated to single source of truth
- Now everything has consistent schemas
